## Graceful Degradation and Service Workers

Graceful degradation is the practice of designing a web application so that it continues to function at a reduced but acceptable level when something goes wrong, such as a lost network connection or an unsupported browser feature. Service workers support this directly: by intercepting network requests and serving cached responses when the network is unavailable, service workers allow the app to remain usable offline instead of presenting the user with a blank page or error.
